== Type Systems

[[type-systems]]
.Type Systems
image::mindmaps/type-systems/type-systems.svg[Type Systems, 100%]

* Unterschiedliche Typen bei verschiedenen Konzeptinstanzen: Beim Konzept "+" basiert der Typ auf den Typen der beiden Argumente. 1 + 2 ist IntType, aber 1 + 2.5 ist DoubleType
* Typkonzepte im Programmtext notwendig bei expliziter Deklarierung: Wenn IntType ein Konzept ist, kann es z.B. zur Variablen-Deklarierung genutzt werden: `IntType x = 4711`.
