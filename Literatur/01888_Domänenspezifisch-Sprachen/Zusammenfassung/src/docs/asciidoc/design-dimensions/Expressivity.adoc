* Leicht schwächere Definition: DSL ist meistens ausdrucksstärker, aber nicht in manchen Spezialfällen. DSL ist optimiert für den allgemeinen Fall, aber braucht Code in allgemeinerer Sprache für Spezialfälle.

* Konflikt zwischen Expressivity und Coverage: Je ausdrucksstärker und kürzer Programme geschrieben werden können, je geringer ist die Abdeckung der Domäne, da man immer inflexibler wird.
