[[language-reuse-uml]]
.Language Reuse
image::language-reuse.png[Language Reuse, 100%]

* Zwei unabhängige Sprachen l2 & l1
* Zwei Fragmente f2 & f1, wobei f2 von f1 abhängt
* lA erweitert l2 (Language Extension). D.h. alle Konzepte aus l2 sind in lA verfügbar.
* lA referenziert l1 (Language Referencing). D.h. Konzepte aus l1 können in lA referenziert werden.
* Wenn lA nun ein Konzept aus l1 erweitert, kann dieses Konzepte aus l1 referenzieren.
* Der Programmierer verwendet dann natürlich lA.
* l2 & l1 bleiben unabhängig von einander und können mit beliebigen anderen Sprachen verbunden werden
* Für die Referenzen in lA gilt
